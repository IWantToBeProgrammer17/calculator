# Aplikasi kalkulator dengan React dan Tailwind CSS

## Latar Belakang dan Fungsi

kalkulator ini merupakan kalkulator scientific yang sederhana. 

Bisa menghitung:
- Operasi matematika sederhana (termasuk penggunaan kurung, kali, bagi, tambah, kurang, dan lain-lain)
- Menghitung eksponen, logaritma. dan akar. Masing-masing bisa menghitung bilangan bulat dan pecahan

## Tech stacks

1. Vite
2. React
3. Tailwind CSS v4
